Medical annotation system

智慧化
醫療標註輔助系統

本計畫開發一個「智慧化醫療標記輔助系統」，因為病人肺癌圖片當中，並沒有標記肺癌位置，因此本計畫先開發一個平台，請醫師協助標記結節位置後，再透過標記的答案，作為模型訓練的答案基礎。

Use functions

點選結節

醫師可在左邊橫切面影像中點選肺結節，並可利用滑動滑鼠滾輪來快速切換左邊畫面中同一位病患的各張CT橫切面影像。操作醫師在結節處點擊滑鼠左鍵兩下後，下方欄位即會顯示出患者編號、結節的X、Y、Z座標，且通過下拉式選單即可選擇標註該結節的類型。

瀏覽結果

提供醫師確認每一位病人結節所圈選位置，可透過點擊按鈕，跳轉至相對應CT圖片以及刪除鍵可讓使用者快速刪除照片，醫生也可以按下修改鍵，立即修正結節種類，並提供備註區域以利醫師提供該病人其他相關資料及訊息。

肺部切割

醫生可透過此功能修正病人的肺臟及結節位置後。系統會自動生成此病人所有圖片中的肺臟結節。左邊為該病人之橫切面原圖，中間為切割肺部的結果，右邊為切割結果與原圖合併的樣子。醫生可以利用滑鼠框出肺部區域遺漏的部分，框完後，下方即會顯示病人編號、張數、起點和終點座標。送出後可以在下方的瀏覽結果瀏覽是否正確，並提供刪除鍵刪除資訊。

確認遮罩

醫師可透過「確認遮罩」功能，將系統自動生成結節的結果，發現有系統標記錯誤處，進行相關說明及註解，系統即可透過模型修正，修正自動標記肺部結節功能之模型。

enter the page

上傳病人

透過「上傳病人」功能，醫師可上傳肺部相關圖片包含橫切面照、全身照之dicom檔、文字報告、及病人編號，系統即可進行肺部及肺部結節自動標記功能，並將相關的文字報告做圖片及病理報告對應。

enter the page

半自動生成遮罩

透過「半自動生成遮罩」功能，醫生上傳橫切面之dicom檔後，系統會自動生成png並顯示在網頁上。當滑鼠移入圖片後，照片會自動放大，以利醫師圈選腫瘤。接著在腫瘤處點擊滑鼠兩下獲取座標後，會在右方的欄位顯示座標資訊，送出座標後，系統即會生成肺部與腫瘤遮罩。

侵襲性與非侵襲性辨識

由於資料不平衡的緣故，此部分使用了四個方法來解決此問題，分別有 1. Imbalanced Data Classification 2. Proposed 3. SMOTE 4. Down Sampling 並搭配Random forest classifier 來預測腫瘤類型。除此之外，也提供了線上上傳dicom檔來辨別腫瘤之侵襲或非侵襲性的功能。
